Install packages:
pip install flask marshmallow sqlalchemy psycopg2-binary
pip install flask_sqlalchemy
pip install flask-restful
pip install flask_jsonpify
pip install flask_script
pip install flask_migrate

Frontend:
run npm i before run npm serve for front end
run frontend command: npm serve

Backend:
run backend command: python main.py
In backend so far only main has been use, other files has not do anything, test.py contain information for database connect to server
