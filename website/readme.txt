Install packages:
Check is npm and flask intalled if you do skip to step 3
1. go to this website to download nodejs and npm:https://www.npmjs.com/get-npm
2. run flask install command using pip: pip install Flask (website instruction:http://flask.pocoo.org/docs/1.0/installation/)
3. install the remainning packages:(if there is any more packages need to be install go ahead and install them.)
pip install flask marshmallow sqlalchemy psycopg2-binary
pip install flask_sqlalchemy
pip install flask-restful
pip install flask_jsonpify
pip install flask_script
pip install flask_migrate

Frontend:
run "npm i" to install the node-packages, then run "ng serve" to start the frontend
run frontend command: "ng serve"
go to http://localhost:5000/, for the website

Backend:
run backend command: "python main.py"
In backend so far only main has been use, other files has not do anything, test.py contain information for database connect to server
